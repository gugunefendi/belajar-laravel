Syarat Menginstall Laravel

Sebelum kamu menginstall Laravel ada beberapa syarat yang dibutuhkan:

  1. PHP sudah terinstall
  2. Composer sudah terinstall
  3. Laravel Installer sudah terinstall
  4. Node.js dan NPM sudah terinstall

Disini ga ada tulisan gimana caranya install PHP, Composer, Node.js ataupun NPM. Kamu bisa cari aja di Google dengan kata kunci cara install PHP, cara install composer dan lain-lain udah banyak tutorialnya.

Oke, saya asumsikan kamu udah menginstall syarat-syarat yang di butuhkan diatas ya..

Kalau PHP dan Composer sudah terinstall, sekarang kita install Laravel Installer melalui composer:

                
                    composer global require laravel/installer

Oke sekarang PHP✅, Composer✅ dan Laravel Installer✅ sudah terinstall.

Sekarang kita buat aplikasi Laravel baru dengan perintah:

                
                    laravel new example-app

Ganti example-app dengan nama aplikasi yang ingin kamu buat misalnya aplikasi-perpus atau aplikasi-karyawan.

Setelah mengetikkan perintah diatas, Laravel Installer akan minta kamu untuk memilih framework untuk testing, database yang digunakan dan starter kita yang kamu inginkan. Pilih sesuai kebutuhan kamu.

Oke sekarang kita masuke ke direktori laravel kita:

                
                    cd example-app

Lalu jalankan:

                
                    npm install && npm run dev
